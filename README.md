--- Azure Speech & Language Studio
Este projeto foi desenvolvido como parte do desafio de projeto da DIO, com foco na aplicação prática das ferramentas de inteligência artificial da Microsoft Azure: Speech Studio e Language Studio.
O objetivo é explorar recursos de análise de fala e linguagem natural, documentando o processo técnico e os aprendizados adquiridos.

--- Objetivos do Projeto
- Aplicar conceitos de IA com foco em linguagem e voz
- Utilizar Speech Studio para transcrição e análise de fala
- Utilizar Language Studio para análise de sentimentos e entidades
- Documentar o processo técnico e os aprendizados
- Compartilhar a experiência via GitHub

--- Ferramentas Utilizadas
- Microsoft Azure Speech Studio  
- Microsoft Azure Language Studio  
- GitHub  
- Python (via API REST, opcional)

--- Etapas Realizadas
-- Speech Studio
- Acesse o Speech Studio
- Crie um projeto de transcrição de fala
- Teste com áudios próprios ou exemplos
- Documente os resultados

-- Language Studio
- Acesse o Language Studio
- Crie um projeto de análise de sentimentos
- Insira textos em português e analise os resultados
- Documente os insights obtidos

--- Sobre o arquivo .env
Este projeto inclui um arquivo .env.example como modelo para configuração das credenciais do Azure. No entanto, o arquivo .env real não foi
preenchido nem incluído no repositório por razões de segurança, conforme boas práticas de proteção de dados sensíveis.

--- Aprendizados

- Aprendi a integrar serviços de voz e linguagem em soluções práticas
- Ganhei familiaridade com ferramentas de IA no Azure
- Reforcei boas práticas de segurança ao lidar com credenciais
